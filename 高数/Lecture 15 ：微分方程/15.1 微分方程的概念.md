## 微分方程的概念

### 微分方程及其阶

表示未知函数及其导数（或者微分）与自变量之间关系的方程称为**微分方程**，一般写成

$$F[x, y, y', \cdots, y^{(n)}] = 0 \text{ 或 } y^{(n)} = f[x, y, y', \cdots, y^{(n-1)}]$$

> 
> 满足两个条件：
> 
> ① 方程；
> 
> ② 含未知函数的导数（微分）
> 
> 研究对象是 $y$，最终求出 $y=y(x)$。


微分方程中未知函数的最高阶导数的阶数称为**微分方程的阶**。如 $y''' - y'' + 6y = 0$ 就是三阶微分方程。
> 对于一个三阶微分方程来说，除 $y'''$ 外，其他均可缺。

### 常微分方程

未知函数是一元函数的微分方程称为**常微分方程**。如 $y''' - y'' + 6y = 0$，$ydx - (x + \sqrt{x^2 + y^2})dy = 0$。
>一元函数说明只包含一个自变量



### 线性微分方程

形如 $a_n(x)y^{(n)} + a_{n-1}(x)y^{(n-1)} + \cdots + a_1(x)y' + a_0(x)y = f(x)$ 的微分方程称为 **$n$ 阶线性微分方程**，其中 $a_k(x) (k=0, 1, 2, \cdots, n)$ 都是自变量 $x$ 的函数，$a_n(x) \neq 0$。当 $a_k(x) (k=0, 1, 2, \cdots, n)$ 都是常数时，又称方程为 **$n$ 阶常系数线性微分方程**。若右端函数 $f(x)$ 恒为零，则称方程为 **$n$ 阶齐次线性微分方程**，否则称其为 **$n$ 阶非齐次线性微分方程**。

### 微分方程的解

若将函数代入微分方程，使方程成为恒等式，则该函数称为**微分方程的解**。微分方程解的图形称为**积分曲线**。

### 微分方程的通解

若微分方程的解中含有的**独立常数的个数等于微分方程的阶数**，则该解称为**微分方程的通解**。
> 经过任何恒等变形都不能使常数个数减少。如 $y = e^x(C_1 + C_2x)$。但 $y = C_1 \sin x + C_2 \cdot 2 \sin x = (C_1 + 2C_2) \sin x = C \sin x$，这里的 $C_1, C_2$ 就不独立。

> 
> 此处的常数并非一定是任意常数，可能是在一定范围内取值的常数。

### 初始条件与特解

确定通解中常数的条件就是**初始条件**。如 $y(x_0) = a_0, y'(x_0) = a_1, \cdots, y^{(n-1)}(x_0) = a_{n-1}$，其中 $a_0, a_1, \cdots, a_{n-1}$ 为 $n$ 个给定的数。确定了通解中的常数后，解就成了**特解**。


## 辨析微分方程的属性

**1. 阶（Order）**

- **定义**：微分方程中**未知函数的最高阶导数的阶数**称为微分方程的阶。
- **辨析**：如果方程中出现的最高阶导数是 $y'$（即一阶导数），例如 $\frac{dy}{dx} = f(x)g(y)$ 或 $y' + p(x)y = q(x)$，这就是**一阶微分方程**。如果方程中出现了 $y''$（二阶导数），例如 $y'' + py' + qy = 0$，这就属于**二阶微分方程**（也统称为高阶微分方程）。

**2. 线性（Linear）与 非线性（Non-linear）**

- **辨析**：资料中反复提及了“一阶线性微分方程”、“二阶常系数线性微分方程”等概念。所谓“线性”，是指方程中的**未知函数 $y$ 及其所有的导数（$y', y''$ 等）都只以“一次方”的形式出现**，并且**不存在 $y$ 和 $y'$ 相乘，或者以 $y$ 为自变量的复合函数**（比如 $y^2$, $\sin y$, $e^y$ 等形式）。
- **例如**：$y'' + py' + qy = f(x)$ 中 $y'', y', y$ 的指数都是 1，所以是线性的。如果方程中出现了 $yy'$ 或者是 $y^3$，那就是非线性的。

**3. 常系数（Constant Coefficient）与 变系数**

- **辨析**：这一属性主要针对线性微分方程中 $y$ 及其各阶导数前面的“系数”。如果这些**系数都是常数（实数）**，这就叫常系数微分方程。
- **例如**：对于二阶线性微分方程 $y'' + py' + qy = 0$，资料明确指出，当**“其中 $p, q$ 为常数”**时，它被称为“二阶常系数齐次线性微分方程”。反之，如果前面的系数是关于 $x$ 的函数（例如一阶线性方程中的 $p(x), q(x)$），那就是变系数的。考研中高阶方程重点考查的是“常系数”。

**4. 齐次（Homogeneous）与 非齐次（Non-homogeneous）** 这里需要特别注意，“齐次”这个词在微分方程中有两种完全不同的语境：

- **语境一：针对线性微分方程（看等号右边的非齐次项）** 这是最常见的分类。当把含有 $y, y', y''$ 的项都放在等号左边后，观察等号右边仅含自变量 $x$ 的项（自由项）：
    - **齐次**：如果等号右端为 $0$，例如 $y'' + py' + qy = 0$，这就叫**齐次线性微分方程**。
    - **非齐次**：如果等号右端是一个**不恒为 $0$ 的已知函数 $f(x)$**，例如 $y'' + py' + qy = f(x)$，这就叫**非齐次线性微分方程**，其中 $f(x)$ 被称为非齐次项。
- **语境二：针对一阶方程的一种特殊题型（齐次型）** 资料中提到了一阶方程中的“齐次型微分方程”。这种方程的形式是 $\frac{dy}{dx} = \varphi(\frac{y}{x})$，即右端可以化为仅由 $\frac{y}{x}$ 构成的函数。它的解法通常是作变量代换 $u = \frac{y}{x}$。这里的“齐次”与线性方程中的齐次是两个概念。

**综合举例：** 以考研最核心的题型 **$y'' - 2y' + y = e^x$** 为例，我们对它进行综合属性辨析：

1. 最高阶导数是 $y''$，所以是**二阶**。
2. $y'', y', y$ 的次数都是一次，且没有互相相乘，所以是**线性**的。
3. $y', y$ 前面的系数是 $-2$ 和 $1$，是常数，所以是**常系数**的。
4. 等号右边是 $e^x$，不为 0，所以是**非齐次**的。 因此，它的全称就是**“二阶常系数非齐次线性微分方程”**，其求解套路是先求对应齐次方程的通解，再去寻找自身的一个特解。
