##### **定理**

- 定理2(唯一性)　给出数列 ${x_n}$，若 $\lim_{n\to\infty}x_n=a$（存在），则 $a$ 是唯一的。

- 定理3(有界性)　若数列 ${x_n}$ 极限存在，则数列 ${x_n}$ 有界。
  
- 定理4(保号性)　设 $\lim_{n\to\infty}x_n=a>b\ ( < )$，则存在 $N>0$，当 $n>N$ 时，有 $x_n>b\ ( < )$。若数列 ${x_n}$ 从某项起有 $x_n\ge b\ (\le)$，且 $\lim_{n\to\infty}x_n=a$，则 $a\ge b\ (\le)$，其中 $b$ 为任意实数，常考 $b=0$ 的情形。
> 
> 脱帽解法：$\lim_{n\to\infty}x_n>a\ ( < ) \Rightarrow x_n>a\ ( < )$（严格不等）。
> 
> 戴帽解法：$x_n\ge a\ (\le) \Rightarrow \lim_{n\to\infty}x_n\ge a\ (\le)$（非严格不等）。
> 
> $\to$ 例如：$\lim_{n\to\infty}x_n>1 \Rightarrow x_n>1$，$x_n>-1 \Rightarrow \lim_{n\to\infty}x_n\ge -1$。
