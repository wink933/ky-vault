### 不定积分概述

**定义：**
- **不定积分**是微分运算的逆运算。给定一个函数 \( f(x) \)，其不定积分表示为 \( F(x) + C \)，其中 \( F(x) \) 是 \( f(x) \) 的原函数，\( C \) 为任意常数。

**基本性质：**
1. **线性性质**：
   - 若 \( c \) 为常数，则 \(\int cf(x)dx = c\int f(x)dx\)。
2. **积分与微分互逆**：
   - 如果 \( F'(x) = f(x) \)，则 \(\int f(x)dx = F(x) + C\)。

**基本积分公式：**
1. 常数函数的积分：\(\int c dx = cx + C\)。
2. 幂函数的积分（\( n \neq -1 \)）：
   \[
   \int x^n dx = \frac{x^{n+1}}{n+1} + C
   \]
3. 指数函数的积分：
   \[
   \int e^x dx = e^x + C
   \]
4. 三角函数的积分（部分常见）：
   - \(\int \sin x dx = -\cos x + C\)。
   - \(\int \cos x dx = \sin x + C\)。

**换元法：**
- 通过变量替换简化积分过程。设 \( u = g(x) \)，则 \( du = g'(x)dx \)。

**分部积分法：**
- 适用于乘积形式的函数，公式为：
   \[
   \int u dv = uv - \int v du
   \]

### 考研重点与典型题目

#### 单选题
1. 下列关于不定积分的说法错误的是：
   A. 不定积分是微分运算的逆运算。
   B. 常数函数的不定积分为 \( cx + C \)。
   C. 幂函数的不定积分为 \( \frac{x^{n+1}}{n+1} + C \)（\( n \neq -1 \)）。
   D. 指数函数 \( e^x \) 的不定积分为 \( x e^x + C \)。

**答案：D**

#### 填空题
1. 幂函数 \( x^n \) （\( n \neq -1 \)）的不定积分是 ______。
2. 指数函数 \( e^x \) 的不定积分为 ______。

**答案：**
1. \( \frac{x^{n+1}}{n+1} + C \)
2. \( e^x + C \)

#### 简答题
1. 请解释分部积分法的原理，并给出一个具体例子。
   
**答案：**  
分部积分法是通过将乘积形式的函数转换为更易处理的形式来求解不定积分。公式为：
\[
\int u dv = uv - \int v du
\]
例如，计算 \( \int x e^x dx \)：
- 选择 \( u = x \)，则 \( du = dx \)。
- 选择 \( dv = e^x dx \)，则 \( v = e^x \)。

应用分部积分法：
\[
\int x e^x dx = x e^x - \int e^x dx = x e^x - e^x + C
\]

### 常见错误与避坑点

1. 忽略常数 \( C \)，导致答案不完整。
2. 换元法中变量替换不正确，导致积分过程复杂化或无法求解。
3. 分部积分时选择不当，导致计算困难。
