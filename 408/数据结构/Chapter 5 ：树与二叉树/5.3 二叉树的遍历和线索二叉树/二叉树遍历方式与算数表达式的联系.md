### 与算数表达式的联系

| 二叉树遍历    | 访问顺序      | 对应表达式记法   | 上例结果                         |
| -------- | --------- | --------- | ---------------------------- |
| **前序遍历** | 根 → 左 → 右 | **前缀表达式** | `* + a b - c d`              |
| **中序遍历** | 左 → 根 → 右 | **中缀表达式** | `a + b * c - d`（⚠️需括号才能保持原意） |
| **后序遍历** | 左 → 右 → 根 | **后缀表达式** | `a b + c d - *`              |
#### 中序遍历如何加括号？
运算优先级按照树的深度来判断，树越深，优先级就越高。
通常情况下，每遇到一个运算符，就要加上一对括号，这样一直加到最后，再把最外围的括号去掉。
